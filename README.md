📌 Team Task Manager (SaaS Project)

A full-stack role-based task and project management system built with React (Frontend) and Flask (Backend API).
Deployed using Vercel (frontend) and Railway (backend).

🚀 Live Links
🌐 Frontend (Vercel): https://your-vercel-link.vercel.app
⚙️ Backend (Railway): https://zoological-art-production-640b.up.railway.app
🧠 Features
👤 Authentication
User Signup / Login
JWT-based authentication
Role-based access (Admin / Member)
📁 Project Management
Admin can create projects
Admin assigns users to projects
Members can only view assigned projects
📌 Task Management
Create tasks inside projects
Assign tasks to users
Mark task as:
Pending
Done
Delete tasks (Admin only)
👑 Admin Panel
View all users
Add members to projects
Manage project access
📊 Dashboard
Total tasks
Completed tasks
Pending tasks
Overdue tasks
🏗️ Tech Stack
Frontend
React.js
Axios
React Router DOM
Backend
Flask
Flask-JWT-Extended
Flask-SQLAlchemy
Flask-CORS
Deployment
Vercel (Frontend)
Railway (Backend)
